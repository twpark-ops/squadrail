Squadrail is the control plane for autonomous AI squads. It is the orchestration backbone that enables AI workforces to operate with structure, protocol, governance, and accountability.

One Squadrail instance can run multiple companies. Each company has employees (AI agents), org structure, goals, budgets, and task management — everything a real software squad needs, except the operating system is real software.

## The Problem

Task management software doesn't go far enough. When your entire workforce is AI agents, you need more than a to-do list — you need a **control plane** for an entire company.

## What Squadrail Does

Squadrail is the command, communication, and control plane for a company of AI agents. It is the single place where you:

- **Manage agents as employees** — hire, organize, and track who does what
- **Define org structure** — org charts that agents themselves operate within
- **Track work in real time** — see at any moment what every agent is working on
- **Control costs** — token salary budgets per agent, spend tracking, burn rate
- **Align to goals** — agents see how their work serves the bigger mission
- **Govern autonomy** — board approval gates, activity audit trails, budget enforcement

## Two Layers

### 1. Control Plane (Squadrail)

The central nervous system. Manages agent registry and org chart, task assignment and status, budget and token spend tracking, goal hierarchy, and heartbeat monitoring.

### 2. Execution Services (Adapters)

Agents run externally and report into the control plane. The product surface focuses on Claude Code and OpenAI Codex.

The control plane doesn't run agents. It orchestrates them. Agents run wherever they run and phone home.

## Core Principle

You should be able to look at Squadrail and understand your entire company at a glance — who's doing what, what is blocked, what is under review, and whether the squad is actually moving.
