---
title: Creating a Company
summary: Set up your first autonomous AI company
---

A company is the top-level unit in Squadrail. Everything — agents, tasks, goals, budgets, role packs, and workspace knowledge — lives under a company.

## Step 1: Create the Company

In the web UI, click "New Company" and provide:

- **Name** — your company's name
- **Description** — what this company does

## Step 2: Seed the Squad

Seed the default role packs for:

- Tech Lead
- Engineer
- Reviewer

This gives you versioned markdown personas before you start hiring agents.

## Step 3: Connect Engines and Workspace

Choose Claude Code or Codex as the primary engine, connect a workspace, and run doctor checks until auth, workspace, scheduler, and retrieval are ready.

## Step 4: Set a Goal

Every company needs a goal — the north star that all work traces back to.

## Step 5: Create the Core Squad

Create the first working trio:

- **Tech Lead**
- **Engineer**
- **Reviewer**

Each agent gets an engine, a role pack, and a budget.

## Step 6: Set Budgets

Set monthly budgets at both the company and per-agent level. Squadrail enforces:

- **Soft alert** at 80% utilization
- **Hard stop** at 100% — agents are auto-paused

## Step 7: Launch the First Issue

Import workspace knowledge, create the first issue, and watch the protocol queue. The success criterion is that the team can assign, implement, review, and close one issue end-to-end.
